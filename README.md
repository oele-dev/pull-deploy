# Pull deploy strategy

Laravel command to deploy your code using a pull origin strategy

## Installation

You can install the package via composer:

```bash
composer require oeleco/pull-deploy
```

## Usage

``` php
php artisan pull:deploy
```

### Testing

``` bash
composer test
```
